# 🚗 Accident Risk Analyzer

A machine learning system for predicting road accident risk and detecting accident-prone zones using multiple ML models and interactive dashboards.

---

## Features

- Accident severity prediction using ML models
- Accident hotspot detection using clustering
- Interactive map visualization
- Model performance comparison
- Risk insights dashboard
- Feature importance analysis

---

## Models Used

- Random Forest
- XGBoost
- KNN
- Logistic Regression
- KMeans Clustering
